    Client¶

    The Client table contains client-specific calls as well as tables with APIs for UIs. It is only available on the Client context.

    The "Client character" here refers to the currently-controlled character on the client.

    ClientLib Class¶

    Inherits from Library.

    Events and Hooks¶

    ActiveCharacterChanged (event)¶

    @field PreviousCharacter EclCharacter

    @field NewCharacter EclCharacter? Can be `nil` in GM mode.

    SkillStateChanged (event)¶

    Fired when the active client character enters or exits a skill state.

    @field Character EclCharacter The active client character.

    @field State EclSkillState?

    ViewportChanged (event)¶

    Fired when the size of the viewport changes.

    @field NewSize Vector2

    @field OldSize Vector2

    Methods¶

    CopyToClipboard¶
    function ClientLib.CopyToClipboard(text) -- (Client-only)

    Copies text to the clipboard.

    Some characters, like newlines, will be trimmed out as per the vanilla scripting on the swf.

    Relies on MessageBox.

    @param text string

    GetActiveUI¶
    function ClientLib.GetActiveUI(playerIndex) -- (Client-only)
   -> UIObject?

    Returns the last UIObject the player has interacted with.

    @param playerIndex integer? Defaults to `1`.

    GetCharacter¶
    function ClientLib.GetCharacter(playerIndex) -- (Client-only)
   -> EclCharacter

    Returns the currently-controlled character on the client.

    @param playerIndex integer? Defaults to 1.

    GetDate¶
    function ClientLib.GetDate(utc) -- (Client-only)
   -> DateTime

    Returns the current date and time as an object. Relies on Client.UI.Time

    @param utc boolean Whether to use UTC time. Defaults to false.

    GetDateString¶
    function ClientLib.GetDateString() -- (Client-only)
   -> string

    Returns the current date and time as a string.

    GetGlobalUIScale¶
    function ClientLib.GetGlobalUIScale() -- (Client-only)
   -> number

    Returns the UI scale preference of the user.

    This refers to the UIScaling setting; it is unaffected by resolution.

    GetMousePosition¶
    function ClientLib.GetMousePosition() -- (Client-only)
   -> integer, integer

    Returns the global position of the mouse in the screen space, in pixel coordinates.

    GetProfileGUID¶
    function ClientLib.GetProfileGUID(playerIndex) -- (Client-only)
   -> GUID

    Returns the profile GUID of a player.

    @param playerIndex integer? Defaults to 1.

    GetUI¶
    function ClientLib.GetUI(ui) -- (Client-only)
   -> UI

    Returns the Epip UI table for a UI, if registered.

    @param ui UIObject|integer UI or type ID.

    GetViewportSize¶
    function ClientLib.GetViewportSize() -- (Client-only)
   -> Vector2

    Returns the viewport size.

    IsActiveCombatant¶
    function ClientLib.IsActiveCombatant() -- (Client-only)
   -> boolean

    Returns whether the client's char is currently playing its turn in a combat. TODO this fails while fear is applied.

    Relies on StatusConsole.

    IsCursorOverUI¶
    function ClientLib.IsCursorOverUI(playerIndex) -- (Client-only)
   -> boolean

    Returns whether the cursor is over any UIObject.

    @param playerIndex integer? Defaults to `1`.

    IsHost¶
    function ClientLib.IsHost() -- (Client-only)
   -> boolean

    Returns true if the client is the host of the current session.

    Relies on GameMenu.

    IsInCombat¶
    function ClientLib.IsInCombat() -- (Client-only)
   -> boolean

    Returns whether the client char is in combat.

    IsInDialogue¶
    function ClientLib.IsInDialogue() -- (Client-only)
   -> boolean

    Returns whether the client character is currently in dialogue.

    IsInGameplayState¶
    function ClientLib.IsInGameplayState() -- (Client-only)
   -> boolean

    Returns true if the game is in a gameplay state, or late into loading.

    IsPreparingInfusion¶
    function ClientLib.IsPreparingInfusion() -- (Client-only)
   -> boolean

    Returns true if the client character is preparing a Source Infusion.

    IsUsingController¶
    function ClientLib.IsUsingController() -- (Client-only)
   -> boolean

    Returns whether the client is using a controller.

    WorldPositionToScreen¶
    function ClientLib.WorldPositionToScreen(worldPosition, playerIndex) -- (Client-only)
   -> Vector2

    Translates world coordinates to screen position.

    @param worldPosition Vector3

    @param playerIndex integer? Defaults to `1`.
